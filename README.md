# Wer trackt wen warum wozu – Chrome Extension Tracker Checker

Eine Chrome Extension, die transparent zeigt, welche Tracker auf jeder Website aktiv sind, welche Daten sie sammeln und warum.

---

## ⬇️ Download & Installation

### Neueste Version: v1.2.0

1. Gehe zu [**Releases**](https://github.com/Klotzkette/tracker-sidebar-plugin/releases/latest)
2. Lade `Source code (zip)` herunter
3. Entpacke die ZIP-Datei in einen Ordner deiner Wahl
4. Öffne Chrome und navigiere zu `chrome://extensions/`
5. Aktiviere oben rechts den **Entwicklermodus** (Developer mode)
6. Klicke auf **„Entpackte Erweiterung laden"** (Load unpacked)
7. Wähle den entpackten Ordner aus
8. Die Extension ist jetzt aktiv ✅

---

## Features

- 🔍 Zeigt alle aktiven Tracker auf der aktuellen Seite
- 📊 Erklärt, welche Daten gesammelt werden und warum
- 🛡️ Mehr Privatsphäre mit einem Klick
- 📋 Sidebar-Ansicht mit detaillierten Tracker-Infos
